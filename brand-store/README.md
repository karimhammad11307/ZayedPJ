# ZAYED E-Commerce Platform

This is a Next.js 14 project bootstrapped with `create-next-app`.

## Documentation
All project documentation, including architecture rules, data flow, development phases, and the complete walkthrough, has been moved to the `docs/` folder.

Please see **[docs/project_documentation.md](docs/project_documentation.md)** for full details.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
